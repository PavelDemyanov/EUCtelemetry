EUC Telemetry Visualization

Программа предназначена для извлечения телеметрических данных (скорость, PWM, GPS скорость, мощность, ток, напряжение, заряд батареи, пробег поездки, температура) из популярных программ DarknessBot и WheelLog и наложения их на синий фон (хромакей) для дальнейшей интеграции телеметрических данных на ваше видео, снятое во время поездки или с камер insta360, GoPro, iPhone или любой другой камеры.

Как экспортировать телеметрические данные

Из DarknessBot:

  1. Откройте DarknessBot и перейдите в раздел "Trips"
  2. Выберите нужную "Trip"
  3. Нажмите кнопку экспорта
  4. Сохраните CSV файл поездки в удобное место

Из WheelLog:

  1. Откройте WheelLog и перейдите в раздел
  2. Дополнительные шаги
  3. Дополнительные шаги
  4. Дополнительные шаги

Работа с программой

  1. Откройте CSV файл
  2. Выберите желаемые настройки FPS, разрешения (FullHD или 4K) и кодека
  3. Настройте отображение элементов телеметрии на видео
  4. Запустите процесс создания видео на синем фоне (Chroma Key)
  5. Перейдите в раздел "My Projects" для скачивания готового видео

Наложение на видео с вашей поездки с iPhone, insta360, GoPro и других камер

  1. Вы можете использовать любой видеоредактор с поддержкой удаления хромакея, например Davinci Resolve
  2. Откройте видео с вашей камеры в Davinci Resolve
  3. Наложите сверху видео, созданное программой euctelemetry.app
  4. Примените эффект 3D keyer к видео от euctelemetry.app
  5. Совместите телеметрические данные с вашей поездкой на таймлайне
  6. Экспортируйте видео с желаемыми настройками качества
